# PayPo - Tu Cupo Digital

PayPo es una aplicación de crédito digital para bodegas y tiendas de confianza. Permite a los usuarios pagar "fiado" usando cupos pre-aprobados.

## Características

- **Cupo PayPo**: Visualización clara del crédito disponible y deuda.
- **Billetes Digitales**: Sistema de pago arrastrable (Drag & Drop) intuitivo.
- **Tienda de Confianza**: Drop zone para pagar en la bodega asociada.
- **Historial de Cuotas**: Seguimiento de pagos y fechas de vencimiento.

## Cómo Iniciar

1. Instalar dependencias:
   ```bash
   npm install
   ```
2. Iniciar el servidor de desarrollo:
   ```bash
   npm run dev
   ```
3. Abrir en el navegador (usualmente http://localhost:5173).

## Tecnologías

- React + Vite
- Tailwind CSS (Estilos Modernos)
- Framer Motion (Animaciones)
- Lucide React (Iconos)
